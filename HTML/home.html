Como jogar?

Enter - Pressione a tecla Enter para começar a jogar

Seta para cima - Pressione a seta para cima para acelerar

Seta para baixo - Pressione a seta para baixo para frear

Seta para esquerda - Pressione a seta para esquerda para virar à esquerda

Seta para direita - Pressione a seta para direita para virar à direita
